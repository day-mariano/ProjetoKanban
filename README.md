# Trella

O Trella é um projeto de kanban, que foi desafio do módulo 3 da Formacão front-end da Ada Tech.

## Funcionalidades:
- Criar usuário
- Adicionar colulas
- Adicionar tarefas
- arrastar tarefas entre as colunas
- apagar e editar tarefas
- dados salvos no local storage

## Páginas:
- Login
- Cadastro
- Kanban

## Feito em:
- React
- TypeScript
